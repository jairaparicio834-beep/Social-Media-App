Developed a full stack social media app with features like posts, post feeds, likes comments, modals, and authentication using Firebase for database managemnent

## Install Packages
```bash
npm i
```
# Setup .env file 
```bash
NEXT_PUBLIC_API_KEY=
NEXT_PBLIC_AUTH_DOMAIN=
NEXT_PUBLIC_PROJECT_ID=
NEXT_PBLIC_STORAGE_BUCKET=
NEXT_PBLIC_MESSAGING_SENDER_ID=
NEXT_PBLIC_APP_ID=
```

# Start the App

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
